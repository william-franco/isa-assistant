# ISA

ISA – Interface Simbiótica de Aprendizado

Interface: atua como ponte entre o usuário e o sistema.
Simbiótica: coopera com o usuário.
Aprendizado: remete ao aprendizado de máquina, machine learning.


## Installing

rm -rf .venv/

python3.12 -m venv .venv && source .venv/bin/activate

pip install -r requirements.txt

python isa.py
